Mini-clase: Cálculo de área de superficie de cilindros

Al terminar esta mini-clase podrás calcular área lateral y total de cilindros con las fórmulas 2πrh y 2πrh+2πr2, y despejar r o h desde un área dada.

Esto se evalúa mucho en PAES en problemas de latas, tubos y estanques.

Idea clave

Áreas del cilindro

La siguiente figura muestra un cilindro recto con radio r y altura h: la superficie lateral (rectángulo curvo) tiene área Al=2πrh y el área total de un cilindro cerrado es At=2πrh+2πr2.

Si el cilindro está abierto, decide si usas cero, una o dos tapas según lo que diga el problema.

π y datos faltantes

Usa π exacto o una aproximación dada (por ejemplo 3,14) y mantén la misma elección en todo el ejercicio.

Si conoces el área lateral Al, plantea Al=2πrh y despeja dividiendo por 2πh para hallar r, o por 2πr para hallar h.

Trampa PAES

❌ Calcular área lateral usando 2πrh+2πr2 (estás sumando las tapas).

✔ Para área lateral usa solo 2πrh; agrega 2πr2 solo si piden área total de un cilindro cerrado.

Ejemplo resuelto

Una lata cilíndrica cerrada tiene diámetro 8 cm y altura 12 cm. Calcula el área lateral para la etiqueta y el área total, usando π ≈ 3,14.

  1. Paso 1: Identifica datos y partes a considerar

    Datos: diámetro 8 cm, por lo que el radio es 4 cm, y altura 12 cm; como la lata está cerrada, el área total incluye las dos tapas circulares. Distingue que el área lateral solo cubre el costado, para no mezclarla con el área total.

  2. Paso 2: Elige la fórmula de área correspondiente

    Para el área lateral usa Al=2πrh y para el área total de un cilindro cerrado usa At=2πrh+2πr2. Así evitas usar la fórmula de área total cuando te piden solo área lateral o olvidar una de las dos tapas.

  3. Paso 3: Sustituye valores (r, h, π) y calcula

    Sustituye r=4, h=12, π≈3,14: Al=2·3,14·4·12=301,44 cm² y At=301,44+2·3,14·42=401,92 cm²; usa siempre el radio, no el diámetro. Trabaja con π aproximado por 3,14 en todos los pasos para que las aproximaciones sean coherentes.

  4. Paso 4: Si el área es conocida, despeja la incógnita y verifica

    Comprueba que el área total es mayor que la lateral y que la diferencia corresponde a las dos tapas: 401,92−301,44=100,48, que coincide con 2·3,14·42. Así verificas que sumaste las dos bases y no solo una.

Si obtuviste Al=301,44 cm² y At=401,92 cm², vas bien — el punto clave fue distinguir entre área lateral y total usando el radio, no el diámetro.

Mini-clase: Traslación de un punto mediante un vector

Al terminar esta mini-clase podrás trasladar puntos sumando o restando las componentes del vector, para hallar la imagen y el punto original.

Esto se evalúa en preguntas de coordenadas en la PAES M1 de Matemática.

Idea clave

Traslación como suma

Una traslación mueve un punto P=(x,y) usando un vector v=(a,b), sumando horizontalmente y verticalmente: P'=(x+a,y+b).

La figura muestra el plano cartesiano: el eje x es horizontal, el eje y es vertical, y el vector une el punto original con su imagen.

Regla inversa

Si conoces el punto imagen P'=(x',y') y el vector v=(a,b), recuperas el punto original restando: P=(x'−a,y'−b).

Piensa siempre que el vector indica cuánto se mueve el punto, no dónde termina.

Trampa PAES

❌ Responder P'=v como si el vector fuera la nueva posición.

✔ Usar la regla de suma: P'=P+v, sumando las dos coordenadas por separado.

Ejemplo resuelto

Traslada el punto P(−2,3) con el vector v(4,−1). Encuentra la imagen P' y luego recupera el punto original usando el mismo vector.

Usa las reglas P'=(x+a,y+b) y P=(x'−a,y'−b).

  1. Paso 1: Identifica P, P' y el vector v(a, b)

    Lees el punto inicial P(−2,3) y el vector v(4,−1); P' aún no tiene coordenadas. Muchos leen mal el signo de x: aquí x=−2, no x=2.

  2. Paso 2: Decide si buscas imagen P' o punto original P

    Primero calcula la imagen, así que trabajas con P'=P+v, es decir, sumas componente a componente. Para recuperar después P desde P' tendrás que restar, y evitas el error común de sumar dos veces.

  3. Paso 3: Opera la componente en x (suma o resta a)

    Para la imagen, en x tienes x'=−2+4=2. Para volver al punto original, haces la operación inversa x=2−4=−2; si sumaras de nuevo obtendrías 6, típico error de signo.

  4. Paso 4: Opera la componente en y (suma o resta b)

    Para la imagen, en y calculas y'=3−1=2. Para recuperar la y original, restas el vector a la imagen: y=2−−1=3; muchos olvidan cambiar esta coordenada y dejan y igual.

  5. Paso 5: Escribe las coordenadas del punto resultado

    La imagen es P'(2,2) y, usando la operación inversa, recuperas P(−2,3). Verifica mentalmente que aplicar v a P lleva a P' y que restarlo a la imagen te hace volver al punto inicial.

Si obtuviste P'(2, 2) y recuperaste P(−2, 3), vas bien — el punto clave fue distinguir entre sumar para la imagen y restar para el punto original.

Mini-clase: Potencia de una potencia con base racional

Al terminar esta mini-clase podrás simplificar potencias de una potencia con base racional, multiplicando correctamente exponentes enteros y fraccionarios.

Esta habilidad aparece en varias preguntas de simplificación de expresiones con potencias en la PAES M1.

Idea clave

Regla potencia de una potencia

Cuando elevas una potencia a otra potencia, conserva la misma base racional (número o fracción) y multiplica los exponentes: ( ( a b ) n ) m = ( a b ) n · m , y esto vale con exponentes enteros (positivos, negativos o cero) y fraccionarios.

Leer bien la estructura

Identifica siempre qué está dentro del paréntesis: el exponente exterior afecta a toda la potencia interior, no solo a la base ni solo al exponente. Piensa primero en la potencia interna y luego aplica la regla al exponente de afuera, por ejemplo en ( x 2 3 ) 3 2 .

Trampa PAES

Incorrecto: Sumar los exponentes en una potencia de una potencia: ( ( a b ) 2 ) 3 → ( a b ) 2 + 3 .

Correcto: Multiplicar los exponentes: ( ( a b ) 2 ) 3 → ( a b ) 6 .

Ejemplo resuelto

Simplifica paso a paso la expresión 3543-12.

  1. Paso 1: Reconoce la estructura

    Identifica que toda la base racional está entre paréntesis: primero aparece 3543 y luego todo eso está elevado a -12.

    Esta estructura coincide con una potencia de una potencia, no con un producto de potencias, por lo que puedes aplicar la propiedad específica.

  2. Paso 2: Escribe el nuevo exponente como producto

    Aplica la propiedad abnm=abn·m y escribe 3543·-12.

    Es común que se quiera sumar 43 y -12, pero en potencia de una potencia siempre multiplicas los exponentes.

  3. Paso 3: Calcula y simplifica el exponente

    Multiplica las fracciones de los exponentes: 43·-12=-46, porque multiplicas numeradores entre sí y denominadores entre sí.

    Reduce la fracción del exponente a -23; si quedaras en -46, el resultado sería equivalente pero menos claro para comparar con otras expresiones.

  4. Paso 4: Reescribe la potencia con el nuevo exponente

    Usa el exponente obtenido y escribe 35-23 como forma simplificada de la expresión inicial.

    Verifica con una calculadora que la expresión original y 35-23 entregan el mismo valor aproximado; si hubieras sumado los exponentes obtendrías un número distinto.

Si obtuviste 35-23, vas bien — el punto clave fue multiplicar correctamente los exponentes fraccionarios y dejar el exponente en su forma más simple.

  • - ¿Toda la base que se repite está dentro de los paréntesis antes de aplicar la propiedad?
  • - ¿Escribiste el nuevo exponente como un producto (no como suma) y multiplicaste bien las fracciones?
  • - ¿Simplificaste la fracción del exponente y revisaste que el signo final sea correcto?

Mini-clase: Concepto de inecuación lineal y conjunto solución

Al terminar esta mini-clase podrás reconocer inecuaciones lineales sencillas y describir su conjunto solución, comparándolo con el de una ecuación.

Esto se usa en la PAES al traducir condiciones verbales con desigualdades.

Inecuaciones lineales y su solución

¿Qué es una inecuación lineal?

Una inecuación lineal compara dos expresiones con una variable usando signos de desigualdad: menor que, mayor que, menor o igual, mayor o igual.

Ejemplo: ax+b>c dice que el lado izquierdo es mayor que el derecho.

Conjunto solución e intervalos

La solución de una inecuación lineal es el conjunto de todos los números que hacen verdadera la desigualdad, casi siempre un conjunto infinito.

Ejemplo: x>3 son todos los números mayores que 3; x≥3 incluye también el 3, formando un tramo seguido en la recta numérica.

Trampa PAES

❌ Tratar una inecuación como ecuación y buscar solo “el número” que la hace verdadera.

✔ Distingue: x=3 tiene una sola solución; x>3 tiene infinitas (por ejemplo, 4, 5, 3,1).

Ejemplo resuelto

En una promo de cine el descuento se obtiene comprando al menos 3 entradas. Compara esto con pedir exactamente 3 entradas.

  1. Paso 1: Traduce la frase a símbolo

    “Al menos 3 entradas” significa “3 o más”, así que la escribes como la inecuación n ≥ 3 ; usar n = 3 pediría solo exactamente 3.

  2. Paso 2: Distingue ecuación e inecuación por el símbolo

    En n = 3 el símbolo central es “=”, es ecuación; en n ≥ 3 ves ≥, es inecuación con desigualdad.

  3. Paso 3: Describe el conjunto solución de la inecuación

    Para n ≥ 3 el conjunto solución son todas las cantidades que hacen verdadera la frase: 3, 4, 5, 6, ..., infinitos valores, no un solo número.

  4. Paso 4: Verifica probando valores cercanos al límite

    Prueba con n = 2 , n = 3 y n = 4 : solo 3 y 4 cumplen n ≥ 3 , así identificas qué valores pertenecen al conjunto solución; si dijera “más de 3”, escribirías n > 3 y 3 ya no pertenecería.

Si entendiste que la inecuación n ≥ 3 tiene infinitas soluciones y representa 3, 4, 5, 6, ..., vas bien — el punto clave fue pensar en un conjunto de valores.

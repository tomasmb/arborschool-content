Mini-clase: Resolución de problemas con cilindros y prismas generales

Al terminar esta mini-clase podrás calcular área lateral, área total y volumen de cilindros y prismas triangulares en situaciones reales, revisando que los resultados tengan sentido físico.

Estos procedimientos se evalúan frecuentemente en PAES M1.

Idea clave

Elegir área o volumen según el contexto

En un cilindro recto calcula volumen con V=πr2h y área según lo que pida la situación: solo parte curva (área lateral) o parte curva más bases (área total); usa π≈3,14 cuando te pidan un valor numérico.

La figura muestra un cilindro y un prisma triangular con sus dimensiones y superficies laterales y de base sombreadas para distinguir qué parte usar en cada problema.

Prismas triangulares y cuerpos combinados

En un prisma de base triangular primero encuentra el área de la base con Abase=bh2 y luego el volumen multiplicando por la altura del prisma; cuando un objeto mezcla cilindro y prisma, suma o resta los volúmenes o áreas calculados por separado y revisa si el resultado tiene sentido para el tamaño real del objeto.

Trampa PAES

❌ Usar el diámetro directamente en πr2 como si fuera el radio.

✔ Si te dan el diámetro d, primero calcula el radio con r=d2 y recién usa las fórmulas de área o volumen.

Ejemplo resuelto

Calcula el área de lámina para un envase cilíndrico sin tapa de diámetro 8 cm y altura 12 cm, usando π≈3,14.

  1. Paso 1: Identifica el cuerpo y los datos

    Es un cilindro recto sin tapa, así que necesitas área lateral y una sola base. El diámetro es 8 cm, entonces el radio es r=4cm; usa siempre el radio en la fórmula, no el diámetro.

  2. Paso 2: Elige la fórmula adecuada

    Para la lámina sin tapa usa área lateral 2πrh más una base πr2; no sumes dos bases porque arriba queda abierto.

  3. Paso 3: Calcula área o volumen

    Con r=4, h=12 y π≈3,14, el área lateral es 2πrh=2·π·4·12=96π≈301,44 cm². El área de la base es πr2=π·42=16π≈50,24, así que el área total de lámina es 96π+16π=112π≈351,68 cm².

  4. Paso 4: Interpreta y verifica el resultado

    El resultado 351,68 cm² es la lámina para costado y fondo del envase. Es positiva, cercana a 300 cm² y algo mayor que el área lateral sola (301,44 cm²), así que sí agregaste la base y no un área exagerada por usar el diámetro como radio.

Si obtuviste 351,68 cm², vas bien — el punto clave fue sumar área lateral y base usando el radio.

# Pruebas PAES M1 para Conversión a QTI

Este directorio contiene las pruebas que se convertirán a formato QTI 3.0 XML.

## Estructura

- `raw/` - PDFs originales (solo lectura, no modificar)
- `procesadas/` - Pruebas en proceso de conversión
- `finalizadas/` - Pruebas completamente convertidas y validadas

## Uso

Ver `app/pdf-to-qti/TRABAJO_PRUEBAS.md` para el flujo de trabajo completo.
